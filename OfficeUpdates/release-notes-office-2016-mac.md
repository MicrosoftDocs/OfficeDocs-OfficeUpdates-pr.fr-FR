# <a name="release-notes-for-office-2016-for-mac"></a>Notes de publication pour Office pour Mac

2016 Office pour Mac est une version d’Office est disponible sous forme d’un fournisseur unique à partir d’un magasin de vente au détail ou via un contrat de licence en volume. Il est pris en charge avec les mises à jour de sécurité et les correctifs, selon vos besoins, jusqu'à ce que le 13 octobre 2020. Le Mac OS pris en charge minimale pour 2016 Office pour Mac est 10.10 (Yosemite).

> [!NOTE]
> - Pour des raisons de sécurité, Microsoft a rendu obsolète l’utilisation de SHA-1. [En savoir plus](https://docs.microsoft.com/en-us/security-updates/SecurityAdvisories/2017/4010323)
> - Pour les notes de publication pour Office 2016 des versions Mac avant septembre 2018, voir [notes de publication pour Office pour Mac](release-notes-office-for-mac.md).  
> - Pour plus d’informations de version pour Office pour Mac, qui est la version fournie avec abonnements Office 365, voir les [notes de publication](release-notes-office-for-mac.md) et [l’historique de mise à jour](update-history-office-for-mac.md) pour Office pour Mac.
> - 2019 Office pour Mac est la version la plus récente est disponible sous forme d’un fournisseur unique. Pour plus d’informations de version pour 2019 Office pour Mac, voir les [notes de publication](release-notes-office-for-mac.md) et [l’historique de mise à jour](update-history-office-for-mac.md) pour Office pour Mac.

Mises à jour pour 2016 Office pour Mac sont disponibles à partir de Microsoft AutoUpdate (MAU). Pour utiliser Microsoft AutoUpdate, démarrer une application Office, telles que Word, puis choisissez **aide** > **des mises à jour**. Si vous êtes administrateur et que vous souhaitez mettre à jour les ordinateurs de votre organisation avec vos outils de déploiement de logiciels existants, vous pouvez télécharger et déployer les packages de mise à jour pour chaque application.
